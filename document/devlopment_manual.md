# Python Machine Learning

## 1. 環境構築と初期設定
- 仮想環境 venv と .gitignore 作成
### 1. 仮想環境構築
    python3 -m venv venv
### 2. 仮想環境を立ち上げる
    source venv/bin/activate
### 3. pip upgrade
    pip install --upgrade pip
### 4. 各種 labrary install & 起動
    pip install jupyterlab
    pip install numpy
    pip install matplotlib
    pip install opencv-python
### 6. jupyter lab 立ち上げ
    jupyter lab
